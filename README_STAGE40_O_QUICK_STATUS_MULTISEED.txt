JUEGO 1.6 - Stage40 O
QUICK STATUS MULTISEED

Objetivo
- Mostrar el resultado multisemilla de vegetacion dentro de ESTADO_RAPIDO_JUEGO.bat.
- Detectar desde el tablero principal si el balance depende de una sola semilla.

Cambios
- ESTADO_RAPIDO_JUEGO.bat lee:
  - stability
  - coverage_min
  - coverage_avg
  - coverage_max
- La seccion VEGETACION muestra estabilidad y rango de cobertura.
- El resumen final incluye la estabilidad multisemilla.

Uso recomendado
- Ejecutar PREVISUALIZAR_VEGETACION_BIOMAS.bat despues de tocar vegetacion.
- Luego ejecutar ESTADO_RAPIDO_JUEGO.bat y revisar multiseed.
