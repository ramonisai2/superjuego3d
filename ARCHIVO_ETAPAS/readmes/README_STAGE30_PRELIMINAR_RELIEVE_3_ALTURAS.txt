STAGE 30 PRELIMINAR - RELIEVE POR 3 ALTURAS

Objetivo:
Probar un cambio grande del generador: dejar de depender de una sola malla ondulada y empezar a dividir el mundo en capas de altura.

Capas nuevas:
- Capa 1 / baja: fondos de lagos, zonas húmedas, valles bajos.
- Capa 2 / media: mundo jugable normal, planos, bosques, desiertos y rutas comunes.
- Capa 3 / alta: mesetas, montañas y accidentes geográficos que tapan parte del paisaje.

Lagos:
- Ahora intentan nacer desde la capa baja.
- El agua se coloca en un nivel bajo, con más profundidad visual.
- El oasis se mantiene como anillo alrededor, no encima del lago.
- Deben aparecer menos como parches pintados y más como agua dentro de zonas bajas.

Mesetas:
- Se mantiene la idea de mesetas altas con paredes más difíciles de escalar.
- Deben existir rampas naturales o caminos menos abruptos.

Rendimiento/carga:
- Se mantiene el sistema de carga/LOD de chunks de la versión anterior.
- Se mantiene la corrección del bug de caída/reaparición.

Qué revisar:
1) Si aparecen lagos más visibles que antes.
2) Si el lago se siente dentro de una zona baja.
3) Si el oasis ya no se come el agua.
4) Si las mesetas siguen funcionando.
5) Si hay errores de altura, caídas, zonas negras o carga pesada.

Posibles errores esperados:
- Algunos lagos pueden quedar demasiado bajos o demasiado concentrados.
- Algunas transiciones entre capa baja/media/alta pueden sentirse raras.
- Puede haber zonas planas grandes; es parte de probar el sistema de capas.
