Stage37 Y - MENU PRINCIPAL

Objetivo:
- Tener una entrada unica para el proyecto.
- Evitar recordar muchos nombres de .bat.
- Separar acciones comunes de menus especializados.

Resultado:
- Se agrega INICIO_JUEGO.bat.
- Permite abrir OpenGL estable.
- Permite abrir OpenGL recomendado.
- Permite verificar entorno Python.
- Permite entrar al menu de presets.
- Permite ver/instalar dependencias.
- Permite configurar o administrar Python guardado.
- Permite abrir preview de terreno y log de movimiento.

Nota:
- No reemplaza los lanzadores existentes.
- No instala dependencias sin confirmacion.
- No abre Vulkan.
- No crea ZIP.
