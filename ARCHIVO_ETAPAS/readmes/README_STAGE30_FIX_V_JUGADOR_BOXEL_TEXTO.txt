STAGE 30 FIX V - JUGADOR BOXEL + TEXTURAS

Objetivo:
- Pulir al jugador principal en tercera persona.
- Mantener la skin, pero combinarla con mas volumen boxel.

Cambios principales:
1) Se mantiene la skin del personaje como identidad visual base.
2) Se agregan piezas boxel encima de la textura:
   - hombros,
   - pechera,
   - cinturon y hebilla,
   - correas,
   - mochila con mas volumen,
   - botas,
   - protectores en piernas,
   - brazaletes,
   - volumen de cabello/capucha.
3) Se mejora la silueta general para que el jugador se vea menos plano.
4) Se conserva animacion basica de caminar y pose de nado.
5) Se copia la skin limpia dentro del proyecto para usarla por defecto.

Idea futura:
- Este sistema se puede reutilizar para NPCs humanos con skins distintas,
  pero manteniendo el mismo lenguaje visual boxel + textura.
