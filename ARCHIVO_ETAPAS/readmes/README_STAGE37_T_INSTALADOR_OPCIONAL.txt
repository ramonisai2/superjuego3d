Stage37 T - INSTALADOR OPCIONAL

Objetivo:
- Facilitar la instalacion de dependencias cuando falten.
- Evitar instalaciones accidentales.
- Conectar instalacion, verificacion y menu de presets.

Resultado:
- Se agrega INSTALAR_DEPENDENCIAS_JUEGO.bat.
- El instalador muestra paquetes y pide escribir SI antes de instalar.
- MOSTRAR_COMANDO_DEPENDENCIAS.bat menciona el instalador opcional.
- VERIFICAR_ENTORNO_JUEGO.bat sugiere el instalador si falta Python valido.
- PROBAR_PRESETS_OPENGL.bat agrega la opcion 12.

Nota:
- El instalador no se ejecuta automaticamente.
- Requiere conexion si pip necesita descargar paquetes.
- No abre el juego.
- No crea ZIP.
