STAGE 28 FIX D - AGUA REDUCIDA + SAVE WORLD + LAUNCHER

Por feedback visual, se frenaron los superlagos experimentales.

Problemas observados:
- El agua grande se leia como una placa plana.
- Al poner agua en laderas parecia flotar.
- El fondo se hundia como agujero.
- La tierra bajo el agua se decoloraba demasiado.

Cambios:
1) Se quitaron superlagos grandes de prueba.
2) Quedan charcas/humedales pequeños y organicos.
3) Ya no se excava un agujero artificial bajo el agua.
4) El color del suelo bajo agua conserva mas la tonalidad del bioma.
5) El agua tiene menos opacidad.
6) Launcher rapido en: JUEGO 1.6/LANZADOR_RAPIDO.bat
7) Se mantiene Save World, menu inicial y pantalla de carga.

Nota de diseño:
Los lagos enormes se dejan para una etapa posterior con sistema de cuencas reales
y malla/shoreline dedicada. Por ahora es mejor agua pequeña que se vea menos rota.
