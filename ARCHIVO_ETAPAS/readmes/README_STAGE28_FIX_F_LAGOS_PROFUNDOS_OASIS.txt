STAGE 28 FIX F - LAGOS PROFUNDOS + OASIS

Cambios principales:
- Los lagos ahora se tallan mas profundo.
- El fondo del lago tiene un suavizado/curva mas visible.
- La superficie del agua intenta sentirse horizontal y calmada, en vez de seguir tanto la ladera.
- Se aumento la teselacion del terreno (subdivisiones) para que el borde del lago y la cuenca se lean mejor.
- Las orillas del lago generan mucho mas pasto, flores, arbustos y arboles.
- Incluso si el bioma base es seco/desertico, alrededor del lago aparece un efecto de oasis.

Notas:
- Esta version prioriza la lectura visual del lago y de sus orillas.
- Aun no es un sistema hidrologico completo, pero ya empuja mas hacia lagos profundos con vegetacion rica.
