STAGE34 F - SLIME CON HUELLA CLARA

Objetivo:
- Hacer que jugador, NPC y slime compartan una huella X/Z mas parecida.
- Mantener al slime mas bajo que los humanoides.
- Darle mas personalidad sin meter sistemas nuevos ni costo pesado.
- Corregir el rastro al morir para que no quede enterrado bajo el suelo.

Cambios:
- Slime visual mas ancho y profundo:
  - cuerpo aproximado X/Z: 1.04 x 0.98
  - sombra aproximada X/Z: 1.04 x 0.96
  - caja debug aproximada X/Z: 1.04 x 0.98
- Patitas separadas con spread X/Z independiente.
- Animacion con salto suave, balanceo lateral y respiracion gelatinosa.
- Cara con pupilas vivas, boca variable y cejas cuando esta agresivo.
- Charco/remanente al morir mas grande y elevado sobre el terreno.

Notas:
- No se cambio el modo recomendado: OpenGL estable sigue siendo el camino jugable.
- No se hizo ZIP.
- Esta etapa es visual/comportamiento, no una optimizacion grande de FPS.
