Stage36 P - DETALLE POR DISTANCIA

Objetivo
- Dibujar menos piezas en entidades lejanas.
- Mantener el detalle completo cerca del jugador.
- Preparar una ruta gradual para LOD de arboles, hierba y terreno.

Cambios
- Se agregan niveles visuales full, mid y far para entidades.
- Slimes lejanos usan un cuerpo mas simple y menos piezas internas.
- Slimes a media distancia conservan forma, pero omiten detalles pequenos.
- NPCs cercanos conservan skin completa.
- NPCs a media distancia ocultan herramienta de espalda y rasgos finos.
- NPCs lejanos usan una silueta humanoide barata con color de profesion.
- Restos de slime se ocultan antes que entidades importantes.

Notas
- No cambia IA, dano, recetas, mochila ni escala boxel.
- Objetivos fijados o NPCs destacados fuerzan detalle completo.
- OpenGL estable sigue siendo la ruta jugable.
- No se genero ZIP.
