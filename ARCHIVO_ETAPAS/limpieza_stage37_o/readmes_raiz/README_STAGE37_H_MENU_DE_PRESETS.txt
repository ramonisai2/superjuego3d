Stage37 H - MENU DE PRESETS

Objetivo:
- Reducir pasos manuales al comparar rendimiento.
- Tener un solo archivo para limpiar log, probar presets y analizar resultados.
- Mantener OpenGL estable como camino jugable principal.

Resultado:
- Se agrega PROBAR_PRESETS_OPENGL.bat.
- El menu permite limpiar el log de presets.
- El menu puede abrir low, balanced y high con sesiones nombradas.
- El menu puede analizar todo el historial o solo la ultima sesion.

Uso recomendado:
1. Abrir PROBAR_PRESETS_OPENGL.bat.
2. Elegir 1 para limpiar el log.
3. Probar 2, 3 y 4 caminando 1-2 minutos en cada preset.
4. Elegir 5 para comparar todos los resultados.

Nota:
- OpenGL sigue siendo la ruta estable.
- Vulkan sigue como laboratorio experimental.
- No se crea ZIP en esta etapa.
